# 5-Week Plan: Academic Writing Quiz Preparation
## Converting Module 1 & 2 Activities to Interactive HTML Pages

---

## Overview

This plan outlines the conversion of Module 1 (Components of Academic Journal Articles) and Module 2 (Paraphrasing, Summarising & Synthesising Skills) activities into interactive HTML pages for Weeks 1-5. These pages will prepare students for the Academic Writing Quiz (AWQ) in Week 6.

**Target Skills for AWQ:**
- Analyse academic texts to summarise, synthesise, and paraphrase key information
- Use academic tone and proper citations and references
- Create outlines with hierarchy (main arguments, sub-points, supporting details)
- Identify key points from journal articles
- Write 300-word summary in 60 minutes

**Time Allowed:** 60 minutes

## Technical Specifications

### HTML Page Structure
Each HTML page should include:
1. **Header:** Week number, topic, learning objectives
2. **Navigation:** Previous/Next buttons, progress tracker
3. **Content Area:** Interactive activities
4. **AI Assistant Panel:** Collapsible sidebar for AI help
5. **Feedback Section:** Real-time feedback display
6. **Progress Save:** Auto-save student progress

### Interactive Elements
- **MC Questions:** Radio buttons with immediate feedback
- **Fill in the blank:** Text inputs with validation
- **Matching:** Drag-and-drop interface
- **Sentence/Paragraph Writing:** Text areas with word count
- **Demos:** Animated, step-by-step guides
- **Timers:** For timed practice exercises

### AI Integration
- Use HKBU Gen AI API (`gpt-4.1` model)
- Real-time feedback on writing
- Plagiarism detection
- Citation validation
- Peer review facilitation
- Personalized tutoring

### Assessment Features
- Self-assessment quizzes
- Peer review system
- Teacher feedback submission
- Progress tracking
- Performance analytics
- Mock AWQ attempts
